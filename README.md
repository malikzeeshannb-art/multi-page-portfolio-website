# Multi Page Portfolio Website

This project is a beginner frontend development project built during my learning journey.

It demonstrates how to structure and build a real multi-page website using only HTML and CSS.

The goal of this project was to understand how professional websites organize pages, layouts, and reusable design systems.

---

## Project Pages

This project includes three main pages:

• Portfolio Page – Displays projects and work samples  
• Case Study Page – Explains a project in detail  
• Contact Page – Allows visitors to get in touch through a contact form

---

## What I Practiced In This Project

• Multi-page website architecture  
• Reusable layout structure (shared header and footer)  
• Global CSS design system using CSS variables  
• Responsive image handling  
• Accessible HTML form structure  
• Organized folder structure for frontend projects

---

## Technologies Used

HTML5  
CSS3

No JavaScript or frameworks were used because this project focuses on mastering core HTML and CSS fundamentals.
